# Security Policy

## Privacy by design

Kintora stores everything **locally** on the user's own computer. It has no
servers, no accounts, and sends nothing over the network. Your data never leaves
your machine.

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅        |

## Reporting a vulnerability

If you discover a security or privacy issue, please report it **privately** first
rather than opening a public issue:

- Use GitHub's **"Report a vulnerability"** (Security → Advisories) on this repo, or
- Open a minimal issue asking for a private contact channel (do **not** include
  exploit details or any personal data in a public issue).

Please allow a reasonable time for a fix before any public disclosure. Thank you
for helping keep Kintora safe.
